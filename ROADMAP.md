# 🗺️ Roadmap

> This roadmap represents the planned features of Shipyard. \
> Each feature is assigned to a release version following **Semantic Versioning (SemVer)**. Once assigned, a feature's identifier is never reused, ensuring stable references throughout the project's lifecycle.
>
> Implementation details are tracked separately by Shipyard's task system.
